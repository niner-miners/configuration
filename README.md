# Niner Miners Core

The nuts and bolts of the UNC Charlotte Official Minecraft server.

## Directory Layout

```
$HOME/
├── api/
│   └── # a Node server running an API
├── core/
│   └── # this repository
└── minecraft/
    ├── data/
    │   ├── logs/
    │   └── ...
    └── server/
        └── # all Minecraft server files
```
